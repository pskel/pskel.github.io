PSkel

A high-performance framework for parallel skeletons.

Introduction

PSkel is a high-performance framework for parallel skeletons. PSkel provides a single high-level abstraction for parallel skeletons on heterogeneous CPU-GPU systems. The current implementation uses parallel skeletons to transparently leverage Open Multi-Processing (OpenMP), Intel Threading Building Blocks (TBB - Intel Corporation), and NVIDIA CUDA (Nvidia Corporation). The current version of PSkel supports both the stencil and map parallel patterns.

Requirements

Dependencies

  • NVIDIA CUDA: PSkel has not been tested with any NVIDIA CUDA version under 5.5.
  • OpenMP: OpenMP is used throughout the PSkel, not only during the skeletons computations.
  • TBB: PSkel can use either OpenMP or TBB for multi-threading the skeletons computations.
  • GAlib: GAlib is used to solve optimization problems required for some autotuning mechanisms.
  • Google Test: Google testing framework is used for unit-testing PSkel.
